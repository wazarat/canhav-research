# Logo Images for About Us Page

This folder contains the institutional logos for the "Trusted & Engaged" section.

## Required Files:
1. `pvara-logo.png` - Pakistan Virtual Asset Regulatory Authority (green gradient logo)
2. `ddqic-logo.png` - Dunin-Deshpande Queen's Innovation Centre (circular logo)
3. `kingston-logo.png` - City of Kingston (blue logo with building)
4. `mic-logo.png` - Mayor's Innovation Challenge (MIC gradient logo)
5. `coindesk-logo.png` - CoinDesk (yellow and black logo)

## Image Specifications:
- Format: PNG with transparent background preferred
- Size: Optimized for web (under 100KB each)
- Dimensions: Will be displayed in 128x80px containers
- Quality: High resolution for crisp display

## Usage:
These logos are displayed in the About Us page trust section to show institutional partnerships and recognition.
